# muon

## include

    <script src="path/to/muon.js"></script>

## compile

This library is compiled with [coffeemill](https://github.com/minodisk/coffeemill) in [Node.js](http://nodejs.org/).

    $ coffeemill -m -j muon -c COPYRIGHT -t test
